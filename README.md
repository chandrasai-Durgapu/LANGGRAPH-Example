# LANGNGRAPH-Example
LangGraph Example Project

# Create Virtual Environment
```bash
python -m venv py-001
```

## Activate Virtual Environment
```bash
py-001\Scripts\activate
```

## Install Dependencies
```bash
pip install -r requirements.txt
```
